# js2wasm-baselines

Test262 baseline data for [js2wasm](https://github.com/loopdive/js2wasm) — generated artifacts, not hand-edited.

Updated automatically by CI on every merge to `main`.

## Files

- `test262-current.json` — latest baseline summary (pass/fail/compile_error counts)
- `test262-current.jsonl` — latest baseline full results (one JSON object per test)
- `test262-results.jsonl` — latest full run results
- `runs/index.json` — trend history (appended each run, used by the dashboard)
